**What is an API? (Simple Explanation)**  

An **API** (Application Programming Interface) is like a menu in a restaurant. The menu lists the dishes you can order and tells you what you need to ask for, but you don’t see the kitchen or how the food is actually prepared. Similarly, an API lists the actions a program can request from another program (or service) and the information it needs to provide. You send a request using the API, the other program does the work behind the scenes, and then returns the result to you.  

*Everyday example:* A weather‑app on your phone uses a weather‑service API. The app asks the API, “What’s the forecast for New York?” The API fetches the data from its servers and sends the forecast back to the app, which then shows it to you. You never see the complex data‑gathering process; you just get the weather information you asked for.
